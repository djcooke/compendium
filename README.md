## Local Development

Pre-requisites:
  - [Ruby](https://www.ruby-lang.org/en/documentation/installation/)
  - [Jekyll & Dependencies](https://jekyllrb.com/docs/installation/)

To run:

```sh
  bundle exec jekyll serve
```

To build search index:

```sh
  ./scripts/build-search-data.bash
```
